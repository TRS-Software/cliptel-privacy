1. Scope

This Privacy Policy applies to the Chrome browser extension ClipTel – Clipboard to Telegram and all associated data processing. ClipTel is browser-side software with no cloud infrastructure of its own. There is no ClipTel server that processes or stores user data. The Extension routes data exclusively between the user's device and Telegram's Bot API service.

3. General Principles and Legal Bases

3.1 Data Minimisation

ClipTel is built on the principle of data minimisation (Art. 5(1)(c) GDPR). Only data strictly necessary to the core function — bidirectional clipboard transfer via Telegram bot — is processed.

3.2 Legal Bases

  • Art. 6(1)(b) GDPR (contract performance) — for configuration data (token, chat ID)
  • Art. 6(1)(a) GDPR (consent) — for transmission of clipboard contents to Telegram; obtained via onboarding checkbox; withdrawable at any time by uninstalling or using "Disconnect & reset"
  • Art. 6(1)(f) GDPR (legitimate interests) — for technically necessary local storage (e.g. last update ID)

4.2 Processing per Chrome Permission

storage
Stores all configuration data listed above locally in the browser. Not accessible to websites or other extensions.
clipboardRead
Reads clipboard content exclusively on explicit user action (button click, keyboard shortcut Ctrl+Shift+C, or context menu). Content is then transmitted to the Telegram Bot API and stored locally in the history.
clipboardWrite
Writes incoming text messages from the user's mobile device to the clipboard automatically on receipt.
notifications
Shows desktop notifications confirming sent/received text (max. 100-character preview). The bot token never appears in notifications.
alarms
Creates a repeating background alarm (min. 30s, max. 60s) for Telegram polling. Technically required because Manifest V3 service workers cannot maintain persistent connections.
offscreen
Creates an invisible background page on demand to perform clipboard read/write operations on behalf of the service worker (Chrome MV3 requirement). Contains no personal data; released after each operation.
contextMenus
Adds "Send to phone (ClipTel)" to the browser's right-click menu on selected text. Selected text is transmitted on click.
host_permissions: https://api.telegram.org/*
The only domain contacted by the Extension. No other websites, services, or APIs are accessed. The Extension does not read or modify any pages visited by the user.

5. Data Transmission to Telegram

5.1 Recipient and Data Categories

Telegram Messenger Inc. 71–75 Shelton Street · London, WC2H 9JQ · United Kingdom
Privacy Policy: telegram.org/privacy

Transmitted data: clipboard text content · bot token (in API request URL) · chat ID · IP address of the user's PC (technically unavoidable).

⚠️ Not end-to-end encrypted: Telegram cloud chats via Bot API use server-side encryption. Telegram can technically access the content of transmitted messages. Do not send passwords or highly sensitive data via ClipTel.

Transmission to Telegram servers is always over HTTPS (TLS). The onboarding flow draws attention to this via a mandatory confirmation checkbox.

7. International Data Transfers

7.1 Telegram (UK / UAE)

Telegram is based in the UK, for which an EU adequacy decision exists (28 June 2021, Art. 45 GDPR). For Telegram's operational base in Dubai (UAE), no adequacy decision exists; transfers are based on Art. 49(1)(a) GDPR (explicit consent) following the onboarding notice. Users are informed that the UAE may not provide an equivalent level of data protection to EU law.

7.2 Google Chrome Web Store

Installation via the Chrome Web Store is subject to Google's Privacy Policy and outside the responsibility of TRS Software.

8. Automated Decision-Making and Profiling

ClipTel does not carry out automated decision-making within the meaning of Art. 22 GDPR and creates no user profiles. No analytics, tracking, or usage statistics are collected.
